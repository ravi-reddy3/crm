# Pulse CRM

Starter CRM application built with plain Node.js, HTML, CSS, and JavaScript.

## Features

- Dashboard with contact, pipeline, and task metrics
- Lead management with search and stage updates
- Deal pipeline board with inline stage editing
- Task list with completion tracking
- Recent activity feed
- Seeded local JSON data for quick demos

## Run

```bash
node server.js
```

The app starts on `http://localhost:3010` by default.
